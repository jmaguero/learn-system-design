# 🌐 Networking Fundamentals

## Key Concepts

### DNS (Domain Name System)
- Translates domain names to IP addresses
- Hierarchy: Root → TLD → Authoritative
- TTL (Time to Live) controls caching

### TCP vs UDP
| Feature | TCP | UDP |
|---|---|---|
| Reliability | Guaranteed delivery | Best-effort |
| Ordering | Ordered | Unordered |
| Speed | Slower | Faster |
| Use Case | HTTP, emails | Video streaming, DNS |

### HTTP/HTTPS
- HTTP: Stateless request-response protocol
- HTTPS: HTTP + TLS encryption
- HTTP/2: Multiplexing, header compression
- HTTP/3: Based on QUIC (UDP-based)

### Latency vs Throughput
- **Latency**: Time for one request to complete (ms)
- **Throughput**: Number of requests per second (RPS/QPS)
- Often a trade-off: optimize for one impacts the other
