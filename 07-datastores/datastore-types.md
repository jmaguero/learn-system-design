# Datastore types

## Key-value
Fast lookups by key; great for sessions, caching, counters.

## Document
Flexible JSON-like documents; good for rapidly evolving schemas.

## Wide-column
Optimized for high write throughput and time-series like access patterns.

## Graph
Optimized for relationship traversals.
